NodeTodo App
---------------------------------------------
simple app to add/edit/delete "TODOs"
mark TODO as complete
add a file attachment to a TODO
one person cannot access TODOs of another

for initial setup:
  - npm init
  - npm install express --save-dep (save as dependency)
                ejs --save-dep
                body-parser --save-dep
                momgoose --save-dep

uses "mLab.com" for mongo DB:
  acct login is "bnipp/mlMs..."
  db user is "test/test"


access as "localhost:3000/api/setupTodos" to setup an initial set of TODO items
